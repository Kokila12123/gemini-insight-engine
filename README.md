# Gemini Insight Engine

Gemini Insight Engine is an AI-powered data analysis application that uses the Gemini 3 API to transform raw datasets into clear, actionable insights.

## About the Project
This project allows users to upload structured data (such as CSV files) and receive AI-generated analysis including trends, anomalies, risks, and recommendations. The goal is to make data-driven decision-making accessible to both technical and non-technical users.

## Features
- Upload CSV datasets
- Detect trends and anomalies
- Generate actionable recommendations
- AI-powered reasoning using Gemini 3
- Simple and interactive web interface

## Tech Stack
- Python
- Streamlit
- Gemini 3 API
- Pandas

Demo video: https://github.com/Kokila12123/gemini-insight-engine

## How to Run Locally
1. Install dependencies:
pip install -r requirements.txt

2.Run the application:
streamlit run app.py
